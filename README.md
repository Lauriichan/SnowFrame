## Snowframe

Snowframe is a library collection which allows to modularly build a application lifecycle.

## Maven

```xml
<properties>
  <snowframe.version>1.0.0</snowframe.version>
</properties>

<repositories>
  <repository>
    <id>lauriichan-release</id>
    <name>Lauriichan Maven Repository</name>
    <url>https://maven.lauriichan.me/release</url>
  </repository>
</repositories>

<dependencies>
  <dependency>
    <groupId>me.lauriichan.snowframe</groupId>
    <artifactId>[SnowFrame Module Id]</artifactId>
    <version>${snowframe.version}</version>
  </dependency>
</dependencies>
```
